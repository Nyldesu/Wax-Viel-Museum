##Wax Veil – User Guide##

Introduction

Welcome to Wax Veil, a low-poly horror game where you explore a haunted museum, collect missing mask fragments, and face a mysterious enemy. This guide will help you get started with installation, controls, and gameplay tips.

---

How to Use the Zip File of the Build

1. Download the ##Wax Veil build zip file##.
2. Extract the zip file to a folder on your computer.
3. Open the extracted folder and double-click the executable (`.exe`) file to start the game.
4. Ensure your system meets the minimum requirements for smooth gameplay.

---

## **Controls**

### **Interaction Keys**

| Key   | Action                                         |
| ----- | ---------------------------------------------- |
| **E** | Open/close doors, pick up objects              |
| **B** | Reload stored battery one by one               |
| **N** | Cover the enemy’s eyes to stop their movements |
| **F** | Turn flashlight on/off                         |

### **Movement**

| Input          | Action                              |
| -------------- | ----------------------------------- |
| **W, A, S, D** | Move forward, left, backward, right |
| **Mouse**      | Look around / control camera        |

## **Goal**

* Collect all the missing **mask fragments** and return them to the enemy.
* **Always keep the enemy in your view**; it moves when not seen.
* Complete the game as fast as possible — a **time tracker** records your progress.

---

## **Enemy Behavior**

* The enemy only moves when the player is **not looking** or is **outside the player camera’s view**.
* Stay aware and plan your movements carefully.

---

## **Pause Menu**

* **Slider Settings**:

  * Enemy Movement Speed
  * Mouse Sensitivity
  * Audio Volume
* **Restart Game**

## **Tips for Players**

* Use the **flashlight strategically** — it consumes battery.
* Use **N** to stop the enemy movement when all fragments are collected.
* **Look at the enemy frequently** to prevent it from advancing.
* Explore rooms thoroughly to find all mask fragments.
* Keep an eye on the **time tracker** for a faster completion score.
